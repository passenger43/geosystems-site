Features

Core calculations for optical leveling

Nivelir covers common field workflows and keeps elevation data organized by project.

Instrument setup

Calculate height of instrument from a known benchmark elevation and staff reading.

Elevation stakeout

See the target staff reading and get a clear raise/lower prompt during detailed stakeout.

Slope calculations

Compute design elevations from a starting height, line length and gradient.

Height survey

Enter a staff reading, calculate the point elevation and save it into the project library.

Projects

Keep each job in a separate project so benchmarks and working data stay clean.

Elevation library

Store benchmarks, design elevations and measured points inside the project.

How access works

  • The trial period lasts 30 days after the first app launch/activation.
  • After the trial ends, PRO access is required for working calculations.
  • Projects and saved data remain available for viewing.
  • PRO access is purchased for 1, 6 or 12 months and does not renew automatically.
  • A new period is purchased only when the user chooses it.
