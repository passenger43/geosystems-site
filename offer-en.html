Public offer

Fixed-term paid access to the functionality of GeoSystems studio mobile applications.

1. General provisions

This document is a public offer issued by sole proprietor Ilya Maltsev, Tax ID 434592932969, Registration No. 311434501800091 (the “Provider”), and sets out the terms for paid access to GeoSystems studio mobile applications.

Payment for the selected access term means full and unconditional acceptance of this offer by the user.

2. Terms

  • Application — a GeoSystems studio mobile app for which access is purchased.
  • Trial period — temporary free access for testing the app in real work.
  • PRO access — paid access to app functionality for a selected fixed term.
  • User — a person using the app and purchasing PRO access.
  • Device — the user's mobile device whose identifier is used to activate access.
  • Payment service — the payment service used to complete the transaction.

3. Subject of the offer

The Provider grants the User a trial period and, after it ends, the option to purchase PRO access to the selected mobile application for the term shown during checkout.

The app name, access term, price and relevant functional scope are shown on the Provider's website and/or inside the application before payment.

4. Contract formation

The agreement is considered concluded when the User pays for the selected PRO access term.

Acceptance of the offer includes the following actions:

  • selecting an access term in the application;
  • opening the payment page;
  • completing the payment by an available method.

5. Access provision

After the payment service confirms a successful payment, the Provider's server activates PRO access for the User's device.

The access term is determined by the selected plan and is counted from the activation or extension time recorded by the Provider's server.

Access is granted for the device identifier used when the order was created. Access recovery in non-standard situations is reviewed by support when technically feasible and when the necessary confirmation data is available.

6. Price and payment

Current prices are shown on the Provider's website and/or inside the application. Payment is made electronically using the methods available on the payment page.

PRO access is provided without automatic renewal. A new term is purchased only when the User chooses to do so.

7. User responsibilities

  • use the app and PRO access for their intended purpose;
  • not attempt to bypass access controls, interfere with the server or the payment flow;
  • provide the information reasonably needed to verify a payment or access issue when contacting support.

8. Provider responsibilities

  • activate paid access after payment confirmation;
  • maintain the access verification mechanism within reasonable technical limits;
  • review user requests related to payment and access;
  • update app functionality and future plan terms without reducing an already-paid access period.

9. Requests and refunds

Requests regarding payment, erroneous charge, non-activation of access or refunds should be sent to geosystemsstudio@yandex.ru.

Additional information is available on the “Payment, access and refunds” page.

10. Personal data

Processing of information needed for the website, applications, support and access verification is described in the Privacy policy.

11. Changes to this offer

The Provider may update this offer. The updated version applies to relationships arising after publication, unless the new version states otherwise.
