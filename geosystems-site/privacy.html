Политика конфиденциальности

ОфертаКонфиденциальностьОплата и возвраты

GeoSystems Studio уважает конфиденциальность пользователей и обрабатывает только те данные, которые необходимы для работы приложений, проверки доступа, оплаты, поддержки и улучшения качества приложений.

1. Какие данные могут обрабатываться

  • технический ID устройства для лицензирования и проверки доступа;
  • название приложения, версия приложения, язык интерфейса, дата установки и дата последней активности;
  • информация о статусе доступа, сроке действия доступа и количестве проверок доступа;
  • технические данные, связанные с ошибками, подключением прибора, командами и ответами тахеометра, если такая логика используется в приложении;
  • адрес электронной почты и текст обращения, если пользователь сам пишет в поддержку.

2. Для чего используются данные

  • для активации и проверки PRO-доступа;
  • для обработки оплаты и возвратов;
  • для поддержки пользователей;
  • для диагностики ошибок, Bluetooth/USB-соединения и работы с тахеометрами;
  • для улучшения приложений, интерфейса, расчётов и стабильности.

3. Полевые и проектные данные

Координаты, проекты, DXF-файлы, точки съёмки, отметки и другие рабочие данные пользователя обычно хранятся на устройстве пользователя. Приложения не предназначены для массовой передачи проектных файлов на сервер GeoSystems Studio, если пользователь сам не отправляет данные в поддержку.

4. Передача данных третьим лицам

Данные могут передаваться платёжным сервисам, хостинг-провайдерам, сервисам инфраструктуры и иным техническим подрядчикам только в объёме, необходимом для работы приложений, оплаты, лицензирования и поддержки.

5. Хранение и защита

Данные хранятся столько, сколько необходимо для работы доступа, выполнения обязательств, поддержки пользователей и соблюдения требований законодательства. Разработчик принимает разумные технические меры для защиты данных от несанкционированного доступа.

6. Обращения пользователей

По вопросам обработки данных можно написать на адрес: geosystemsstudio@yandex.ru.
