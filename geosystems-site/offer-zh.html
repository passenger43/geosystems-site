公开要约

服务条款隐私政策付款与退款

本文件是 GeoSystems Studio 按以下条件订立协议的公开要约。用户安装应用、使用应用、申请访问权限、付款或以其他方式使用 GeoSystems Studio 应用和服务，即表示接受本条款。

1. 定义

  • 应用 指 GeoSystems Studio 发布的移动应用，包括 TS Go!、Nivelir 以及其他应用。
  • 用户 指安装或使用应用的个人。
  • PRO 访问 指在所选期限内访问应用付费功能，且不自动续费。
  • 设备 ID 指用于将访问权限绑定到特定设备的技术标识符。

2. 协议标的

GeoSystems Studio 授予用户使用应用的普通非独占许可。可用功能、免费限制、付费功能和访问期限由具体应用、付款页面或开发者官方信息确定。

3. 访问权限

  • PRO 访问按用户选择的期限提供。
  • 访问权限不是自动续费订阅。期限结束后，用户可自行重新购买访问权限。
  • 访问权限可绑定到设备 ID。用户付款前应仔细核对设备 ID。
  • 付款确认后，访问权限可自动或人工激活。

4. 价格和付款

访问价格显示在应用、付款页面或 GeoSystems Studio 官方材料中。付款通过接入的支付服务处理。支付服务确认付款成功后，付款义务视为完成。

5. 应用使用

  • 用户自行负责原始数据、坐标、高程、全站仪或水准仪设置、项目文件、DXF 文件和现场测量结果的正确性。
  • 应用是用于测量、施工、放样、现场测量、施工控制和检查的辅助工具。用户应根据工作要求、规范和安全规则核查结果。
  • 禁止修改应用、绕过访问检查、分发修改版本或侵犯开发者权利。

6. 责任限制

对于因原始数据错误、仪器设置错误、项目错误、Bluetooth/USB 连接故障、设备故障、用户错误、设备或操作系统问题以及第三方服务导致的错误，GeoSystems Studio 不承担责任。

7. 更新

开发者可以发布更新、修改界面、添加、修改或删除功能、修复错误并继续开发应用。部分功能可能根据现场测试和用户反馈进行完善。

8. 退款

退款流程见“付款与退款”页面。退款请求将根据原因、访问状态和实际使用情况单独审查。

9. 联系方式

有关访问、付款和应用使用的问题，请联系：geosystemsstudio@yandex.ru。
