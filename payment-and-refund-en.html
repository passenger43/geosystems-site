Payment and support

Payment, access and refunds

How PRO access is paid, when it activates and how to contact support in case of an issue.

1. What the user pays for

After the trial period, the user pays for PRO access to the functionality of a GeoSystems studio mobile application for the selected fixed term. The current term and price are shown before payment.

The trial period is provided so the user can test the app in real work. After the trial ends, PRO access is required for working calculations; created projects and saved data remain available for viewing.

2. Payment flow

  1. The user selects a PRO access term inside the app.
  2. The app opens a secure payment page.
  3. The user pays using an available payment method.
  4. The payment service sends a technical payment confirmation to the Provider's server.
  5. The server activates or extends PRO access.

3. Access activation time

Access is activated automatically after the server receives confirmation of a successful payment. In most cases this happens shortly after payment.

If access does not appear within a reasonable time, the user should contact support and include the information needed to locate the payment.

4. No automatic renewal

PRO access is purchased for a fixed term and does not renew automatically. After the term ends, a new access period is purchased only if the user decides to do so.

5. What to include in a support request

  • an email address for the reply;
  • approximate payment date and time;
  • payment amount;
  • device identifier or other support information shown in the app;
  • a clear description of the issue.

6. Refund-related requests

Requests related to mistaken payment, technical non-activation of access and refund questions are reviewed individually through support.

Please send such requests to geosystemsstudio@yandex.ru and describe the situation as precisely as possible.

7. Contact

Email: geosystemsstudio@yandex.ru
